# Python学习管理工具使用指南

## 🚀 快速开始

### 方法1：在GitHub Codespaces中运行
1. 点击仓库的 "Code" → "Codespaces"
2. 创建新的Codespace
3. 在终端中运行：
```bash
python learning_tools.py
```

### 方法2：在本地运行
1. 确保安装Python 3.6+
2. 下载 `learning_tools.py` 文件
3. 在终端中运行：
```bash
python learning_tools.py
```

## 📊 功能说明

### 1. 学习进度跟踪
- 记录每次学习的时间、科目和主题
- 数据自动保存到 `learning_data.json`
- 查看学习历史记录

### 2. 目标管理
- 设定具体的学习目标
- 设置截止日期
- 跟踪目标完成情况

### 3. 番茄工作法
- 25分钟专注 + 5分钟休息
- 可自定义时间和循环次数
- 提高学习效率

### 4. 时间块规划
- 科学的每日时间分配
- 根据不同时段安排合适的学习内容
- 优化学习节奏

### 5. 学习方法库
- **费曼学习法**：通过教学加深理解
- **主动回忆法**：强化记忆效果

## 💡 使用建议

1. **每日记录**：养成记录学习进度的习惯
2. **目标导向**：先设定明确目标再开始学习
3. **时间管理**：结合番茄工作法和时间块
4. **方法交替**：根据不同内容选择合适的学习方法

## 🔧 自定义修改

你可以修改代码中的：
- 番茄钟时长
- 时间块安排  
- 学习方法参数
- 数据存储方式
